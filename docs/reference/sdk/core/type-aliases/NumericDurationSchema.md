# Type Alias: NumericDurationSchema

> **NumericDurationSchema** = `BaseNumberSchema`\<[`Duration`](../enumerations/ValueHintType.md#duration)\> & `HintDiscriminant`\<[`Duration`](../enumerations/ValueHintType.md#duration)\> & `object`

A schema representing a return value or object property that is provided as a number,
which Coda should interpret as a duration. The given number should be an amount of days
(fractions allowed).

## Type Declaration

| Name | Type | Description |
| ------ | ------ | ------ |
| `maxUnit?` | [`DurationUnit`](../enumerations/DurationUnit.md) | The unit to use for rounding the duration when rendering. For example, if using `DurationUnit.Days`, and a value of 273600 is provided (3 days 4 hours) is provided, it will be rendered as "3 days". |
| `precision?` | `number` | A refinement of [DurationSchema.maxUnit](../interfaces/DurationSchema.md#maxunit) to use for rounding the duration when rendering. Currently only `1` is supported, which is the same as omitting a value. |
