makeReferenceSchemaFromObjectSchema

  • makeReferenceSchemaFromObjectSchema(schema: GenericObjectSchema, identityName?: string): GenericObjectSchema
    • Defined in schema.ts:401

    Convenience for creating a reference object schema from an existing schema for the object. Copies over the identity, id, and primary from the schema, and the subset of properties indicated by the id and primary. A reference schema can always be defined directly, but if you already have an object schema it provides better code reuse to derive a reference schema instead.
